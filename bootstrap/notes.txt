pour changer les couleurs: text-primary text-success text-danger etc..

pour changer les couleurs des bg: bg-primary etc..

les formulaires: form-group pour les balises div à l'interieur de form
 et form-control pour les inputs 

 pour mettre tous les inputs dans la meme ligne : form-inline dans la balise form

 on peut ajouter des couleurs de bord pour les input (généralement avec du js) avec les class has-success has-error etc..

 pour agrandir la taille des inputs: ajouter form-control-lg à form-control (dans la v3 c'est input-lg)

pour mettre des checkbox ou/et des radio dans la meme ligne on ajoute form-check-inline à form-check
